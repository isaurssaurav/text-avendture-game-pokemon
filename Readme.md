## Game Feature
1. Player Level, Player can go through tutorial
2. Two Maps, Player can collect two badges : Water and Fire
3. Catch Pokemon, Select Pokemon, View Pokemon, Regain Pokemons health
4. More than 10 Pokemons
5. Sound Effects
